# Universal Video Exporter

클로드 디자인 애니메이션을 MP4로 변환합니다.
ZIP, HTML 파일 모두 지원. Stage 형식이 아닌 일반 애니메이션도 변환 가능.

## 설치 방법 (한 번만 하면 됨)

1. 이 폴더를 원하는 곳에 저장하세요

2. 명령 프롬프트(cmd)를 열고, 이 폴더로 이동하세요
   ```
   cd 이_폴더_경로
   ```

3. 아래 명령어를 순서대로 실행하세요
   ```
   npm install
   npx playwright install chromium
   ```

## 사용 방법

1. 명령 프롬프트에서 서버를 시작하세요
   ```
   npm start
   ```

2. 브라우저에서 http://localhost:4747 을 여세요

3. ZIP 또는 HTML 파일을 드래그해서 업로드하세요

4. 영상 길이, 해상도, 프레임레이트를 설정하고 변환하세요

5. 완료되면 MP4 다운로드 버튼이 나타납니다

## 두 가지 모드

- **Stage 모드**: Stage 기반 프로젝트는 자동 감지되어, 
  영상 길이와 해상도가 자동으로 설정됩니다.

- **범용 모드 (CDP)**: Stage가 아닌 일반 CSS/JS 애니메이션은 
  CDP 가상 시간 기술로 변환됩니다. 
  이 경우 영상 길이를 직접 입력해야 합니다.

## 종료하기

- cmd 창에서 Ctrl + C를 누르면 서버가 종료됩니다
